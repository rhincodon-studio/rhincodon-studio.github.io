PSI Exam System Requirements (macOS)

Important:
請務必在進行任何設定前，先執行 PSI 官方系統檢查工具：
👉 https://syscheck.bridge.psiexams.com/

此工具會檢查：攝影機、麥克風、網路、解析度、OS、禁止程式等
能提早發現可能造成 CKA / CKAD / CKS 考試失敗 的環境問題。

Note:
本文件根據 PSI 官方規範重新整理而成。
詳細內容請參考原文：
https://helpdesk.psionline.com/hc/en-gb/articles/4409608794260-PSI-secure-browser-and-Chrome-Extension-System-Requirements

PSI Exam System Requirements for macOS

(For CKA / CKAD / CKS Candidates)

本文件整理 PSI Secure Browser 與 PSI Chrome Extension 官方規範，並特別針對 macOS 使用者提供最佳實務流程。
建議在考前 48 小時內完整檢查一次。

1. Supported Operating Systems（macOS）​

PSI Secure Browser 與 Chrome Extension 支援以下 macOS：

macOS 版本Secure BrowserChrome Extension備註
Sonoma (14.x)✔ 支援✔ 支援建議版本
Sequoia (15.x)✔ 支援✔ 支援最新支援
Tahoe (26.x)✔ 支援（2025/10 起）❔ 未公布PSI 宣布將支援
Ventura (13.x)✔ 支援至 2025/11~12✔ 支援將終止支援
Monterey (12.x)❌ 不支援❌ 不支援自 2024/12/02 終止

若使用 macOS Monterey（12.x），將無法進行考試。

2. Display & Resolution Requirements​

  • 最低解析度：1024×768
  • 建議解析度：1920×1080（PSI 官方強烈推薦）

macOS Retina 高解析度建議在考試前調整為：

系統設定 → 顯示器 → 解析度 →「更多空間（1920×1080）」

3. Memory, Storage & Hardware​

資源要求
RAM最低 8 GB，建議 16 GB
硬碟空間500 MB 以上（用於下載 Secure Browser）
相機必須可進行 360° 室內環境掃描
麥克風必須全程開啟（禁止耳機、AirPods）
揚聲器建議使用電腦內建喇叭
外接設備不可有額外攝影機、麥克風、螢幕（建議拔除）

筆電使用者需手持筆電轉圈掃描整個房間。
桌機使用者需準備外接可移動之 USB 攝影機。

4. Internet Requirements​

  • 最低 750 kbps 上下行
  • 推薦 3 Mbps 以上
  • 建議 30 Mbps 以上（最佳）

請在考前測試：

  • https://video-diagnostics.twilio.com/
  • https://speed.cloudflare.com/

禁止​

  • VPN
  • Proxy
  • 公司網路（企業 Firewall 會阻擋 PSI 連線）

5. macOS 必做設定（非常重要）​

以下為 PSI 官方針對 macOS 的必要操作：

🔧 必須關閉 Bluetooth​

防止被判定使用無線耳機（多數考試禁止耳機）。

🔧 關閉所有非必要程式​

以下程式一定要關閉：

  • Teams
  • Zoom
  • Slack
  • Discord
  • LINE / WeChat
  • Dropbox / Google Drive / OneDrive
  • Notion（會偵測剪貼板權限）
  • 防毒軟體（若掃描功能影響效能）

🔧 停止所有自動同步​

尤其是：

  • iCloud Drive
  • Dropbox Sync
  • Google Drive File Stream

🔧 拔除所有外接設備（建議）​

  • 外接螢幕
  • USB 攝影機
  • USB 麥克風
  • USB 擴充座
  • 任何額外 USB 驅動裝置

（PSI 在 macOS 上對多攝影機、多麥克風特別敏感。）

6. Testing Software: Secure Browser​

Secure Browser 必須注意：​

  • 請務必重新下載最新版
  • 不可從 Terminal 啟動
  • 不可存在殘留舊版本（需清除）

建議刪除：

~/Library/Application Support/PSI
Applications/PSI Secure Browser.app

再重新下載官方版本。

7. Chrome Extension 模式（若你不是使用 Secure Browser）​

若使用 Chrome Extension：

  • 需使用最新 Chrome
  • macOS 14 / 15 完整支援
  • 仍需能執行 360° 室內環境掃描
  • 仍不可使用耳機
  • 仍需拔掉多餘攝影機與麥克風

8. Room & Desk Requirements​

必備​

  • 光線足夠
  • 臉部要清楚
  • 桌面乾淨
  • 只有一台電腦
  • 門窗需關閉

禁止​

  • 任何紙張、筆記、書本
  • 手機（僅能在驗證身份時短暫使用）
  • 平板
  • 第二台螢幕
  • 小孩、嬰兒、寵物
  • 其他人進出房間

如果有人進入房間，考試可能會 立即中止。

9. Troubleshooting (macOS)​

❗ 相機無法偵測​

  1. 系統設定 → 隱私權與安全性 → 相機
  2. 確保已允許：
    • Secure Browser
    • Chrome
    • FaceTime
  3. 測試：
    • https://webcamtests.com
    • FaceTime App

❗ 麥克風無法偵測​

  1. 系統設定 → 隱私權與安全性 → 麥克風
  2. 確保只啟用一個輸入設備
  3. 測試：
    • https://www.onlinemictest.com

❗ Secure Browser 變慢 / 當掉​

  • 關閉防毒軟體
  • 關閉所有同步程式
  • 重啟 mac
  • 清空 RAM（建議至少 50% 可用記憶體）

10. Pre-Exam Tutorial（強烈建議）​

PSI 提供 Tutorial 測試，可先驗證環境：

  • 會檢查攝影機
  • 會檢查麥克風
  • 會檢查禁止程式
  • 會檢查網路與連線穩定度

請至少在正式考前一天測試一次。

11. 48-Hour Checklist（專為 macOS 考生）​

系統​

  • macOS 14 或 15（不是 Monterey）
  • 重開機
  • 關閉 Bluetooth
  • 關閉 VPN

網路​

  • 速度至少 30 Mbps
  • 不使用公司網路
  • 測試：https://video-diagnostics.twilio.com/

介面與設備​

  • 調整 1920×1080 顯示
  • 拔除所有外接設備
  • 只保留一台攝影機、麥克風

環境​

  • 桌面乾淨
  • 房間無人
  • 打光足夠（可見臉部）

軟體​

  • 關閉 Teams/Zoom/Slack/LINE/Discord
  • 關閉 Dropbox/Google Drive/iCloud
  • 關閉防毒軟體（考完再開）

測試​

  • 完成 PSI Tutorial 測試
  • 確認相機可以進行 360° 室內掃描

12. Final Notes​

使用 macOS 考 CKA / CKAD / CKS 完全可行，只要符合以下三點：

  1. 使用 Sonoma 14 或 Sequoia 15
  2. 關閉 Bluetooth、拔除額外設備
  3. 網路穩定 + 做過 Tutorial 測試
