外掛大腦：讓 AI 成為你的第二大腦，而不是替你思考

April 4, 2026 · 4 min read
balnibarbian
balnibarbian

每次開新對話，AI 都不記得你是誰。你要重複解釋背景、重複說明偏好，就像每次看診都要從頭填病歷。

「外掛大腦」是一個解決這個問題的系統：讓 AI 能讀取你的知識庫，記住你是誰、在做什麼、喜歡什麼。

立即下載

GitHub: rhincodon-studio/obsidian-brain-template

點擊 Use this template → 建立你的私人知識庫

設計哲學：AI 是電動牙刷，不是毒品​

電動牙刷讓你刷牙刷得更乾淨，但你還是要自己刷。它不會替你思考，只是放大你的能力。

這個系統的設計目標：

電動牙刷毒品
你的邏輯，AI 執行AI 替你決定
用越久，你越清楚自己用越久，越依賴
拔掉 AI，知識還在停用就崩潰

如果你發現自己「沒有 AI 就不知道怎麼整理筆記」，這個系統就失敗了。

核心功能​

弱結構，強智能​

不用記複雜的分類規則。丟進 Inbox，執行 /intake，AI 會幫你分類。

你負責：隨手記錄
AI 負責：分析內容 → 判斷歸屬 → 自動分類

自我進化，但你是審核者​

AI 會靜默學習你的偏好，但你確認後才生效。

Identity 相關的變更（使命、目標、信念）一定需要審批。AI 無法在你不知情的情況下改變你的「北極星」。

6 個核心技能​

指令作用
/journal建立今日工作日誌
/intake整理 Inbox，自動分類
/reflect週度覆盤，聚合洞察
/maintain健康檢查
/digest處理 AI 觀察到的偏好
/writing載入你的寫作風格

適合誰？​

  • 筆記很亂，不知道怎麼整理
  • 想讓 AI 記住自己的偏好
  • 學生：每天記學習進度，累積成可複用筆記
  • 工作者：讓 AI 協助管理項目和知識

下載與安裝​

  • GitHub 網頁
  • 命令行
  1. 前往 rhincodon-studio/obsidian-brain-template
  2. 點擊綠色按鈕 Use this template → Create a new repository
  3. 命名為 my-brain，設為 Private（重要：你的日誌會在這裡）
  4. Clone 到本地，用 Obsidian 打開
# 建立你的私人 repo
gh repo create my-brain \
  --template rhincodon-studio/obsidian-brain-template \
  --private

# Clone 到本地
gh repo clone my-brain

# 用 Obsidian 打開
cd my-brain && open .

新手入門指南​

Step 1: 建立你的知識庫​

  1. 前往 obsidian-brain-template
  2. 點 Use this template → Create a new repository
  3. 名稱填 my-brain，選 Private
  4. Clone 到電腦：git clone https://github.com/你的帳號/my-brain.git

Step 2: 安裝必要工具​

  • Obsidian — 免費的 Markdown 編輯器
  • Claude Code — AI 助理（需要 API key）

用 Obsidian 打開 my-brain 資料夾，就能看到整個知識庫結構。

Step 3: 讓 AI 認識你​

打開 Identity/TELOS.md，填寫你的使命和目標：

## 使命
- M0: 成為能夠創造價值的軟體工程師

## 目標
- G0: 2024 年完成 3 個 Side Project

## 核心信念
- B0: 簡單比複雜好

再打開 Identity/CONTEXT.md，寫下你現在在做什麼。

Step 4: 開始使用​

在 my-brain 目錄啟動 Claude Code：

cd my-brain
claude

AI 會自動讀取你的身份設定。試試這些指令：

/journal          # 開始今天的日誌
/intake           # 整理 Inbox（如果有東西的話）
/maintain         # 看看知識庫健康狀態

Step 5: 日常使用​

每天：想到什麼就丟進 Inbox/ 或 journals/

有空時：執行 /intake 讓 AI 幫你分類

每週：執行 /reflect 做週度覆盤

就這樣。不用記規則，AI 會學習你的習慣

模板包含什麼？​

├── Inbox/          # 收集箱
├── Projects/       # 有截止日期的項目
├── Areas/          # 持續維護的領域
├── Resources/      # 可複用的知識
├── Archives/       # 已完成的歸檔
├── Identity/       # 你是誰（AI 讀取）
├── System/         # 系統設定
└── journals/       # 每日日誌

  • 完整的 AI 提示詞（.claude/CLAUDE.md）

  • 範例檔案和詳細說明
